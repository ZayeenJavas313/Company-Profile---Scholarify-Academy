🚀 Scholarify — Seed Data ke Sanity

Gunakan halaman ini untuk memasukkan data awal Mentor, Testimoni, dan Berita ke Sanity Studio.

🔑 Koneksi Sanity

📋 Panduan Singkat

  1. Buka manage.sanity.io → login → buat project baru
  2. Copy Project ID dan isi di form di atas
  3. Buat API token (Settings → API → Add token, pilih akses Editor)
  4. Isi token, lalu klik "Seed Semua Data Sekarang"
  5. Upload foto via Sanity Studio di https://[projectId].sanity.studio
